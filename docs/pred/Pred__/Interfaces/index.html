Module Pred__.Interfaces

the Interfaces module is for interfaces that a type may fulfill, as well as miscellaneous utilities.

Each interface is layed out as follows:

  • Implementation is the module type that an implementer must fulfill.
  • Interface is the module type that a functor which parametrizes over the interface should take.
  • Make is the functor that transforms Implementation into Interface.
