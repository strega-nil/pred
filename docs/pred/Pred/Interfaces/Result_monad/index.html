Module Pred.Interfaces.Result_monad

Result_monad is an extension to regular Monad, with support for an error type.

It should be implemented by any result-like types, instead of the regular Monad, because it gives access to wrap_err.

module type Implementation : sig ... end
module type Interface : sig ... end
module Make : functor (M : Implementation) -> Interface with type error = M.error and type 'a t = 'a M.t
