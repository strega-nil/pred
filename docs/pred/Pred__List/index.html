Module Pred__List

type 'a t = 'a list

type alias

list operations

val hd : 'a list ‑> 'a option
  • Returns the first element of the given list.
  • Returns None if the given list is empty.
val tl : 'a list ‑> 'a list option
  • Returns the given list without its first element.
  • Returns None if the given list is empty.
val nth : int ‑> 'a list ‑> 'a option
  • Returns the n-th element of the given list. the first element is at position 0.
  • Returns None if the list is too short.
  • Raises Invalid_argument: if n is negative.
val rev : 'a list ‑> 'a list
  • Returns the reverse of the parameter.
val flatten : 'a list list ‑> 'a list

Concatenates a list of lists. tail recursive in the size of the overarching list; not tail recursive in the size of each element list.

val concat : 'a list list ‑> 'a list

an alias for flatten

iterators

val map : ('a ‑> 'b) ‑> 'a list ‑> 'b list

List.map f [a0; ... an] applies f to each of am

  • Returns [f a0; ... f an]. tail recursive, unlike the standard library's implementation.
val to_seq : 'a list ‑> 'a Pred.Seq.t

generates a sequence of the elements of the list

val of_seq : 'a Pred.Seq.t ‑> 'a list

collects the elements of the sequence into a list

val fold : 'a ‑> ('a ‑> 'b ‑> 'a) ‑> 'b list ‑> 'a

fold a f [b0; b1; ... bn] is f (... (f (f a b0) b1) ...) bn.
